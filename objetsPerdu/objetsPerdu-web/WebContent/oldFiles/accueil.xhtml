HELLO WORLD !!

En attendant d'avoir l'application du prof, vous pouvez déjà jouer avec ce projet Eclipse !
Vous pouvez ainsi vous entrainer a utiliser git, le reste ne changera strictement rien pour vous, le principe est simple : vous programmez sur Eclipse (projet web-dynamic+maven), vous compilez et lancez sur votre serveur glassfish local et une fois satisfait (et quand il n'y a PLUS de bugs), vous utilisez git push pour envoyer sur le serveur.
Celui-ci devrait normalement automatiquement compiler et mettre en ligne la nouvelle version, cela peut prendre un peu de temps, mais ca se fera... normalement.

En cas de probleme, n'hesitez pas a m'envoyer un mail ou, si je suis sur skype, a m'informer par la. "Un probleme" = le site n'est plus en ligne ou bien git ne marche pas, ou la compilation ne se passe pas. Evitez ABSOLUMENT de push sur le serveur une version qui ne compile pas chez vous surtout, ca m'arrangerait enormement :p

PS : Lorsque vous faites un push, assurez vous d'avoir mis a jour le pom.xml avant, augmentez d'une version (p.ex 0.0.5-SNAPSHOT a la place de 0.0.4-SNAPSHOT), sinon le serveur ne compilera et ne mettra rien a jour puisqu'il croira que c'est toujours la meme version (pensez-y)
